スケジュールされたタスクと定期メンテナンスについて学習します。このモジュールでは、Cron 構文の理解、ユーザーCrontab、システム全体の Cron の場所、Cron 出力のキャプチャ、at を使用した一度限りのスケジューリングについて扱います。実践的なラボと現実世界の課題を通じて、これらの不可欠な Linux スキルを習得します。
